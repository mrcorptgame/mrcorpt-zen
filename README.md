# mrcorpt-zen

Hide distracting UI elements in mrcorpt.js

Inspired by GitHub [Zen Writing Mode](https://github.com/blog/1379-zen-writing-mode), which
hides all but the text, mrcorpt-zen hides all but the game canvas.
Hides the following:

* mrcorpt.js logo
* stats FPS meter
* mrcorpt-inventory-hotbar
* mrcorpt-voila
* dat.gui

Requires kb-bindings with
'zen' keybinding configured (for example to 'F1').
